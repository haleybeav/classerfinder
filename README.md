### Installation

ClasserFinder requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and start the server.

```sh
$ cd classerfinder/client
$ npm install -d
$ node app
```

```sh
$ cd classerfinder/server
$ npm install
$ node app
```
